# web-dukuri

トオリノが主催するWordPress勉強会用のリポジトリです。


## ディレクトリ
- coding : 静的コーディング時のhtml, css, js, 画像など
- theme　: WordPressのテーマファイル

## WordPressの開発設定（Mac想定)
リポジトリをクローンしたら下記を実行し、dist/assets/のシンボリックリンクをテーマファイル内に作成する

`source assetslink.sh`
